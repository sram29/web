Summary: Lattitude vs. Weather

The purpose of this of this project is to compare latitude versus different weather conditions, cloudliness, humidity, and wind speed

Some of the packages and API used are citipy pyton package to gather a sample of city names and their geographic coordinates and OpenWeatherMap API which contains realtime weather data.

The data was then converted to a dataframe using pandas from which plots were created using different relationships like Temperature, Humidity, Cloudiness, and Wind Speed vs the respective city lattitudes.
